    Klasifikasi KNN Menggunakan Scikit-learn (Python)¶

    Apa itu k-Nearest Neighbor (KNN) ?¶

    K-Nearest Neighbors atau KNN adalah algoritma yang berfungsi untuk melakukan klasifikasi suatu data berdasarkan data pembelajaran (train data sets), yang diambil dari k tetangga terdekatnya (nearest neighbors).Dengan k merupakan banyaknya tetangga terdekat.

    Contoh:¶

    Implementasi KNN Pada Dataset Iris Dengan Scikit-Learn¶

    Dataset Iris ini sudah tersedia by default dari Sklearn nya. Dataset berisi tentang 3 macam spesies bunga beserta ukuran petal dan sepal dengan format csv.Jika dijadikan tabel, tiap barisnya menunjukan yaitu : sepal length, sepal width, petal length, dan petal width.Jenis bunganya ada 3 yaitu : setosa, versicolor dan virginica. Ada 50 sampel data tiap jenis bunga,jika ditotal sampel data = 150 sampel data.

    Import Data Dari Sklearn¶

    # python
# import datanya dari sklearn
from sklearn.datasets import load_iris
iris=load_iris() 

print("")
print(iris.target_names)
print("")
print(iris.feature_names)
print("")
print(iris.data)
print("")
print(iris.target)
print("")

    Pertama import data iris terlebih dahulu dari sklearn,untuk penjelasan script yang dimunculkan diatas setelah import sebagai berikut :

    iris.target_names : nama dari data iris.target.

    iris.feature_names : nama dari data fitur iris.

    type(iris.data) : dataset iris.

    iris.target: data target iris.

    Pendefinisian Data Train dan Data Testk¶

    
from sklearn.model_selection import train_test_split
xtrain, xtest, ytrain, ytest=train_test_split(iris.data,iris.target,test_size=0.2)

print (ytrain)
print("")

    xtrain sebagai data x yang ingin kita latih , xtest sebagai data x yang ingin kita tes, ytrain adalah variabel dependen yang dilatih, dan ytest merupakan variabel dependen yang diuji.

    lImport Fungsi KneighborsClassifier Dari Sklearn¶

    from sklearn.neighbors import KNeighborsClassifier
clf=KNeighborsClassifier(n_neighbors=3).fit(xtrain,ytrain)

    Pertama mengiimport Fungsi KneighborsClassifier Dari Sklearn, lalu panggil fungsi klasifikasi clf.

    Menghitung accuracy score¶

    from sklearn.metrics import accuracy_score
print("accuracy (ketepatan) adalah = ", accuracy_score(ytest,clf.predict(xtest)))
print("")

    Script diatas melakukan perhitungan akurasi dengan Confusion matrix. Confusion matrix akan menampilkan nilai akurasi dari y test dengan prediksi.

    Menampilkan Grafik Dengan Matplotlib¶

    import matplotlib.pyplot as plt

accuracy_values=[]

for x in range(1,xtrain.shape[0]):
    clf=KNeighborsClassifier(n_neighbors=x).fit(xtrain,ytrain)
    accuracy=accuracy_score(ytest,clf.predict(xtest))
    accuracy_values.append([x,accuracy])
    pass

import numpy as np
accuracy_values=np.array(accuracy_values)

plt.plot(accuracy_values[:,0],accuracy_values[:,1])
plt.xlabel("K")
plt.ylabel("accuracy")
plt.show()

    Script ini menampilkan grafik dengan Xlabel = K dan Ylabel = accuracy.
